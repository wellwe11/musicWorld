# MusicWorld - Search for music

A simple, easy website to search for artists & upcoming events.

## Features

- Search bar, as well as filter-buttons to help user find artist/events.
- Save artists and see their upcoming events.
- Link to tickets.
- Bio-page for artist.

## Getting Started

### 1. Clone the repo

`git clone https://github.com/wellwe11/musicWorld`

### 2. Install dependencies

`npm install`

### 3. Run the project

`npm run dev`

## Project Context

My very first, full-page website. I've created everything from scratch. Used for practicing design & React-routing.
